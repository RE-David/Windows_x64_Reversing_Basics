Reverse engineering practice repository for x64 Windows binaries.
Each exercise includes:

clean source code
compiled binary with symbols
stripped binary without symbols

The difficulty increases progressively from basic control flow (if, loops, functions, structs) to more advanced anti-reversing techniques such as obfuscation, indirect calls, string encryption, anti-debugging and integrity checks.

Designed for practicing analysis with tools like Ghidra, IDA and x64dbg while comparing how different compilation settings and symbol information affect the reversing process.

IT IS RECOMMENDED TO START WITH THE EXECUTABLES THAT HAVE SYMBOLS
